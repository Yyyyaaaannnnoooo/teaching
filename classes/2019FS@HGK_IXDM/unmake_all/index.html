Data Overfl0w

Das Video zeigt die menschliche Überforderung mit der immensen Menge an Daten, die wir täglich unbewusst aufrufen und beanspruchen. Es werden die 30 meist besuchten Website analysiert und in einer Geschwindigkeit dargestellt, welche das menschliche Gehirn überfordern soll. Nicht die genauen Zahlenwerte sind relevant, sondern das Erkennen der Menge. Fast die Hälfte der Websites stammen aus China. Während die andere Hälfte oft aus den USA oder auch aus Russland stammt. Oft sind es Online-Shops, News-Portale und Soziale Medien. Im Video wird auch ein #traceroute für die Website alibaba.com visuell ausgeführt, welche den Betrachter an die verschiedenen Ortschaften, wo die Daten beim Abrufen der Website reisen lässt. Unterstützt wird die Analyse von Screenshots und Bildschirmaufnahmen, die während dem Unterricht entstanden sind.

Facebookzeichnung

Facebook sammelt Daten von seinen Nutzern. Wenn man in Firefox den Web Developper öffnet, kann man diese Datensammlung beobachten. Unter anderem werden Daten gesammelt, von der Mausbewegung kurz vor (und kurz nachdem) man einen Klick macht. Wenn man die gesammelten Daten im Programm Visual Studio Code mit einem Java Script Code verbindet, der die Daten verwertet, kann man die Spur sichtbar machen, die man auf Facebook hinterlassen hat. Der Code in sketch.js :

function setup(){
  createCanvas(innerWidth, innerHeight);
  beginShape();
  for (const pos of mouse_movement_coordinates) {
  vertex(pos[0], pos[1]);
      }
  endShape()
  const start = mouse_movement_coordinates[0]
  text('start', start[0], start[1]);
  const last = mouse_movement_coordinates_no_scroll.length - 1
  const end = mouse_movement_coordinates_no_scroll[last]
  text('end', end[0], end[1]);
      }
