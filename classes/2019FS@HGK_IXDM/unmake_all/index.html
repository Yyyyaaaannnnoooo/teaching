Facebookzeichnung

Facebook sammelt Daten von seinen Nutzern. Wenn man in Firefox den Web Developper öffnet, kann man diese Datensammlung beobachten. Unter anderem werden Daten gesammelt, von der Mausbewegung kurz vor (und kurz nachdem) man einen Klick macht. Wenn man die gesammelten Daten im Programm Visual Studio Code mit einem Java Script Code verbindet, der die Daten verwertet, kann man die Spur sichtbar machen, die man auf Facebook hinterlassen hat. Der Code in sketch.js :

function setup(){
  createCanvas(innerWidth, innerHeight);
  beginShape();
  for (const pos of mouse_movement_coordinates) {
  vertex(pos[0], pos[1]);
      }
  endShape()
  const start = mouse_movement_coordinates[0]
  text('start', start[0], start[1]);
  const last = mouse_movement_coordinates_no_scroll.length - 1
  const end = mouse_movement_coordinates_no_scroll[last]
  text('end', end[0], end[1]);
      }
